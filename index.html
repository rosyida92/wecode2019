WEEKEND CODE

Weekend Code adalah sebuah event bertemakan pemrograman komputer dan perkembangannya, kerja sama antara Jurusan Rekayasa Perangkat Lunak SMK Negeri 5 Surakarta dengan industri IT yang bergerak dibidang Software Development serta komunitas IT yang ada diwilayah Surakarta dan sekitarnya. Event ini diselenggarakan dengan berbagai bentuk kegiatan diantaranya seminar, workshop, studyclub, dan ujian sertifikasi.

Weekend Code 2019 Classes

Kelas-kelas yang dibuka untuk Weekend Code 2019 kerjasama dengan Software Developer H&N Tech dan Komunitas Andronity. Workshop akan dimulai pada tanggal 2 November 2019 pukul 08.00 WIB - 15.00 WIB bertempat di Lab RPL dan Kelas Axioo
SMK Negeri 5 Surakarta

  1. FRONT-END WEB DESIGN- Kelas ini akan mempelajari cara pembuatan web desain yang berfokus pada tampilan halaman depan/froent-end dengan basic pemrograman menggunakan HTML, CSS, dan Javascript. Framework yang digunakan pada kelas ini adalah Vue.JS. Bertempat di Lab RPL 3 SMK Negeri 5 Surakarta, Jl. LU Adisucipto No 42 Surakarta.
  2. BACK-END WEB DEVELOPMENT - Kelas ini akan mempelajari pembuatan website pada sisi belakang/back-end untuk halaman pengolahan dan pengumpulan data yang dihubungkan dengan database server. Framework yang akan digunakan pada kelas ini adalah Express.JS. Bertempat di Lab RPL 4 SMK Negeri 5 Surakarta, Jl. LU Adisucipto No 42 Surakarta.
  3. ANDROID PROGRAMMING MOBILE APPLICATION - Pembuatan aplikasi untuk perangkat bergerak berbasis Android akan dipelajari di kelas ini. Software yang akan digunakan di kelas ini adalah Android Studio. Tempat workshop akan menggunakan kelas industri Axioo Jurusan RPL SMK Negeri 5 Surakarta, Jl. LU Adisucipto No 42 Surakarta. Note: peserta workshop ini diwajibkan membawa laptop sendiri.
  4. PLATFORMER GAME - Merupakan Game platformer 2D khususnya merupakan genre game klasik yang dinamakan dari gameplay-nya di mana karakter pemain melompat di atas platform untuk melewati rintangan yang ada. Genre ini sering digabungkan dengan genre lainnya seperti shooter, puzzle, maupun action. Software yang digunakan pada kelas ini adalah Construct 2. Bertempat di Lab RPL 1 SMK Negeri 5 Surakarta, Jl. LU Adisucipto No 42 Surakarta.

Vue. JS

Vue js merupakan salah satu framework yang banyak digunakan untuk frontend web design. Hampir sama dengan angular js, vue js menggunakan pendekatan dom yaitu menggunakan id, name atau class dari suatu component html, serta memiliki component routing yaitu vue-router.

Learn More

Express.Js

Express.js adalah satu web framework paling populer di dunia Node.js. Dokumentasinya yang lengkap dan penggunaannya yang cukup mudah, dapat membuat kita mengembangkan berbagai produk seperti aplikasi web ataupun RESTful API. 

Learn More

Android Studio

Android Studio adalah Lingkungan Pengembangan Terpadu (Integrated Development Environment/IDE) resmi untuk pengembangan aplikasi Android, yang didasarkan pada IntelliJ IDEA. Selain sebagai editor kode dan fitur developer IntelliJ yang andal, Android Studio menawarkan banyak fitur yang meningkatkan produktivitas Anda dalam membuat aplikasi Android

Learn More

Construct 2

Construct 2 adalah tools pembuat game berbasis HTML5 yang dikhususkan untuk platform 2D yang dikembangkan oleh Scirra. Construct 2 tidak menggunakan bahasa pemrograman khusus, karena semua perintah yang digunakan pada game diatur dalam EvenSheet yang terdiri dari Event dan Action.
